## REGEX – Python Regular Expression Practice

This project contains simple Python programs created to practice Regular Expressions (regex) using Python’s built-in `re` module.  
It is made for learning and beginner practice.

## Objectives

- Learn basic regex concepts in Python
- Practice `re.search()` and `re.findall()`
- Validate user input
- Extract useful data from text

## Tasks Implemented

- Check if a password has at least one uppercase letter
- Check if a password is strong (uppercase, lowercase, digit, special character, minimum length)
- Count how many times a word appears in text
- Extract email addresses from a string
- Extract email addresses from a paragraph
- Extract numbers from text
- Detect URLs in text
- Detect IPv4 addresses
- Search log lines for error keywords
- Validate phone numbers (123-456-7890 format)

## Requirements

- Python 3.x
- re module (built-in)

## How to Run

- Open VS Code
- Open the project folder in VS Code
- Open any .py (Python) file from the folder
- Open the terminal in VS Code

- Run the program by typing the command below and pressing Enter:
